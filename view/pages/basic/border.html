边框

边框以 .ol-b-作为前缀，主要有： primary, success, warning, danger, pink, purple, pink, light, dark, muted.
其中还需要包含其边框的大小, 以下划线( _ ) 拼接大小: .ol-b-[color]_[number]. 大小主要有： (1, 2, 3, 5, 8, 10)。
如果指定某个方向的边框，则已两个下划线拼接方向： l, t, r, b， 分别为left, top, right, bottom.

圆角

圆角以 .ol-br作为前缀，默认为值为100%, 所以基本都是正方形的物体，表现最佳。
其中还需要包含其圆角的大小, 以下横线( - ) 拼接大小: .ol-br-[number]. 大小主要有： (1, 2, 3, 5, 8, 10)。
如果指定某个方向的圆角，则已两个下划线( __ )拼接方向： tl, tr, br, bl， 分别为top-left, top-right, bottom-right, bottom-left.
完整路径： .ol-br-[number]__[direct]
